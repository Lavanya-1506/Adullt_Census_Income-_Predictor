# Adult Census Income Predictor



## Overview

Adult Census Income Predictor is a Machine Learning web application that predicts whether a person's annual income is greater than $50,000 or less than/equal to $50,000 based on demographic and employment-related information.

The project uses the Adult Census Income Dataset and implements a Random Forest Classification model with a Flask backend and an interactive frontend built using HTML, CSS, and JavaScript.

---

## Features

### Income Prediction

Predict whether an individual's income is:

* Income > 50K
* Income ≤ 50K

### Confidence Score

Displays the prediction confidence percentage generated by the machine learning model.

### Interactive User Interface

Simple and responsive web interface for entering user information.

### Analytics Dashboard

Displays:

* Total Predictions
* High Income Predictions
* Low Income Predictions
* Average Confidence Score
* Prediction Distribution Chart
* Recent Prediction Records

### Machine Learning Model

* Random Forest Classifier
* Data Preprocessing
* Label Encoding
* Model Evaluation
* Feature Importance Analysis
